Возможности v2.0

[NEW] QR Code Generator

Новый модуль! Создавай QR-коды из любого текста или ссылки. Сгенерированный код сохраняется как изображение `qr_code.png` прямо в папке с программой.

System Scanner

Получи полный срез данных о твоей системе: от модели ЦП и загрузки до использования оперативной памяти и дисков. Теперь с более детальным выводом.

Network Tools

Набор сетевых утилит для диагностики. Пингуй хосты, получай публичную информацию об IP-адресах, мгновенно узнавай свой внешний и внутренний IP.

File Manager Pro

Интеллектуальный файловый менеджер. Анализируй содержимое любой папки или автоматически рассортируй хаос в "Загрузках" по категориям.

Data Encryptor

Защити свои данные. Модуль использует криптографически стойкий алгоритм Fernet для шифрования и дешифрования любого текста с помощью секретного ключа.

Password Forge

Создавай несокрушимые пароли. Гибкий генератор позволяет задавать длину и состав пароля, используя криптографически надежный источник случайных данных.

Инструкция по Запуску

Шаг 1: Подготовка

Скачай главный файл VovanToolkit_v2.py с этого сайта. Помести его в пустую папку на своем компьютере.

Шаг 2: Открытие Терминала

Открой терминал (Командная строка или PowerShell на Windows) прямо в этой папке. Самый простой способ на Windows: открой папку, кликни на адресную строку, введи cmd и нажми Enter.

Шаг 3: Установка библиотек

В открывшемся терминале выполни одну за другой следующие команды. Они установят все необходимые "дополнения" для работы скрипта.

pip install rich
pip install psutil
pip install requests
pip install cryptography
pip install qrcode[pil]

Шаг 4: Запуск!

Все готово. Когда последняя библиотека установится, введи команду для запуска инструментария:

python VovanToolkit_v2.py
